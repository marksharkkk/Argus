# Argus - 多Agent智能协作系统

**项目背景**：OpenClaw 作为开源 AI Agent 系统，支持多渠道消息接入与单 Agent 会话管理，但多 Agent 协作编排能力较弱。本项目在 OpenClaw 基础上构建协作编排层，开创了多人多agent的全新协作模式。

**技术栈**：Node.js / TypeScript / Tauri 2.0 / Vue 3 / OpenClaw

**核心工作**：

- **协作树编排引擎**：创新性提出基于图结构的协作拓扑模型，用户可通过可视化拖拽定义节点（人类/AI Agent）与通信链路，实现协作模式的零代码定制，突破传统框架预设协作模式的限制

- **与 OpenClaw 的深度集成**：复用 OpenClaw 的 sessions_spawn/subagents 机制实现 Agent 实例管理，在其单 Agent 能力基础上扩展多 Agent 间通信路由、任务调度与状态监控能力

- **对等协作模型**：设计"人类即节点"架构，人类与 AI Agent 在协作网络中地位对等，可自由定义通信权限与协作链路，打破"人类监督-Agent执行"的传统层级模式

- **记忆传承系统**：构建五层记忆体系（项目/团队/Agent档案/会议/模式库），实现协作经验跨项目迁移与智能推荐

**项目价值**：为 AI 原生组织提供可落地的协作编排方案，实现从"AI 工具化"到"多人多agent协作化"的范式演进
